This is a simple application that plots values of z <- z2 + c for complex numbers z and c

This application has two pages (apart from this one) -

  • /spiral - plots a function for a given point and for some surrounding points.
  • /animate - plots a function for a set of points and animates the resulting images using javascript.

Here are some points to get started with:

Static:
  • 0.1+0.6i
  • -0.05+0.59i
  • -0.042+0.625i
  • -0.092+0.649i
  • -0.5+0.5i
Animations:
  • -0.025706206+0.746410795i -> -0.025706206+0.746410795i 50 frames
  • 0.0743+0.6464i -> -0.1257+0.8464i 400 frames - slow
  • -0.1+0.6264i -> -0.31+0.6264i 1000 frames - very slow
  • -0.314+0.6264i -> -0.317+0.6264i 2000 frames - painfully slow
